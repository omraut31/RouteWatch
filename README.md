RouteWatch is a smart navigation and safety platform that helps users find optimized routes, view real-time traffic heatmaps, locate CNG stations, and send emergency SOS alerts. It’s built with Flask, Leaflet.js, and HERE APIs for seamless user experience.

<h1>✨ Features </h1>
</br>
<h3>🔹 Smart Routing </h3>

Get optimized routes with real-time traffic data.

Supports multiple transport modes (Car, Bike, Walking).

Displays estimated travel time dynamically.
</br>

<h3>🔹 Traffic Heatmap </h3>

Visualize live traffic congestion on roads.

Color-coded density indicators (Green = free, Red = congested).

Helps plan faster alternative routes.

<h3>🔹 CNG Stations Finder </h3>

Find nearby CNG stations with real-time availability.

View station details: status (operational/maintenance), price per kg, operating hours.

Station Owners can manage their stations.

<h3> 🔹 Emergency SOS </h3>

Add emergency contacts.

Send one-click SOS alert with live location to contacts.

Quick assistance in case of accidents or breakdowns.

Frontend: HTML, CSS (Bootstrap / Tailwind), JavaScript

Backend: Python (Flask)

Database: SQLite 

APIs: HERE Maps API, OpenStreetMap, Flask-SQLAlchemy, TwilloAPI

Authentication: Flask-Login
